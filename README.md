## 🎯 プロジェクト概要

### コンセプト

WTB54は、**Auto-KDK（Automatic Keyboard Design Kit）** を使用して設計した、デュアルトラックボール搭載の分割型キーボードです。マウスを使わずにすべての操作をキーボード上で完結できる、効率的なワークスタイルを実現します。

### 特徴

- ✅ **54キー分割レイアウト**（左右各27キー）
- ✅ **デュアル25mmトラックボール**（左右に各1個）
- ✅ **無線接続**（Bluetooth対応、nRF52840搭載）
- ✅ **Kailh Choc V2スイッチ対応**（キーピッチ17mm）
- ✅ **ガスケットマウント構造**（打鍵感向上）
- ✅ **カスタマイズ可能なキーマップ**（ZMK Firmware）
- ✅ **バッテリー駆動**（2600mAh Li-Po × 2）

### 設計思想

1. **マウス不要の作業環境** - デュアルトラックボールでカーソル操作とスクロールを同時実現
2. **無線化による省スペース** - ケーブルレスで机上をすっきり
3. **エルゴノミクス** - 分割レイアウトで自然な手の配置
4. **カスタマイズ性** - Auto-KDKによる柔軟な設計変更が可能

## 📝 組み立て手順

### 事前準備

#### 1. Auto-KDKでの設計

1. [Auto-KDK](https://auto-kdk.pages.dev/)にアクセス
2. キーボードタイプを選択
   - Keyboard Type: Split（分割型）
   - Controller Type: Wireless（無線）
   - Switch Type: Choc V2（キーピッチ17mm）
3. キー配置を編集（54キー、左右各27キー）
4. トラックボールを配置
5. ケースパラメータを調整
6. ファイル出力（zip）

#### 2. 発注

1. **EasyEDA**でPCB基板データを開く
2. **JLCPCB**でPCBA（ダイオード実装）付きで発注
3. **JLC3DP**で3Dプリントケースを発注
4. **遊舎工房**でガスケット用フォームをレーザー加工依頼
5. その他部品をBOOTH、Amazon、AliExpress等で購入

#### 3. 部品到着確認

すべての部品が揃ってから組み立て開始（約2-3週間）

### Phase 1: Chocソケット実装（PCBA済みの場合はスキップ）

**所要時間：1.5-2時間**
 **温度設定：340-350℃**

#### 準備

1. YouTube動画で予習
   - [torabo-tsuki組み立て動画](https://youtu.be/21c7hYJnPYo)
   - Chocソケットのはんだ付け部分を重点的に視聴
2. はんだごてを340℃に予熱
3. 作業スペースを整理
   - 換気扇またはサーキュレーター起動
   - 十分な照明確保
   - 工具を手の届く位置に配置

#### 実装作業

1. PCB基板を裏返す（ソケット実装面を上に）
2. ルーペでソケット取り付け位置を確認
3. **片足予備はんだ法**で実装
   - ソケットの片方パッドに予備はんだを盛る
   - ソケットを配置
   - 予備はんだをこて先で溶かしながらソケットを押さえる
   - 固定されたらもう片方をはんだ付け
4. **5個ずつブロック作業**で進める
   - 1ブロック完了ごとに目視確認
   - 4-5ブロック（20-25個）ごとにこて先クリーニング
5. 左手側27個 → 休憩10分 → 右手側27個
6. 全54箇所の最終確認
   - はんだブリッジなし
   - ソケット浮きなし
   - スイッチ仮挿入テスト

**はんだ付けのコツ：**

- こて先を当てる時間：2-3秒以内
- はんだの形状：富士山型を目指す
- パッドとソケットのピンを同時加熱

------

### Phase 2: コントローラ実装

**所要時間：2-3時間**
 **温度設定：340℃**

#### Step 1: コンスルーの加工

1. 付属の11ピンコンスルーを9ピンに加工（4個すべて）
2. 不要なピン2本をペンチで引き抜く
3. 樹脂の突起をニッパーでカット

#### Step 2: コントローラボードの準備

1. Auto-KDK無線版コントローラのレールをペンチで折り取る
2. 左右2個とも実施

#### Step 3: PCB基板にコンスルーを差し込む

**⚠️ 重要：はんだ付けは不要（差し込むだけ）**

1. キーピッチ17mm用の穴を確認（2つ内側の穴）
2. コンスルー2本の小窓の向きを揃える
   - 小窓側が上（PCB基板の表側）
3. 穴に差し込む
4. 垂直性を確認
5. **PCB基板側ははんだ付けしない**（Auto-KDK設計では不要）

#### Step 4: コントローラを差し込む

**⚠️ コントローラ側もはんだ付けしない**

1. コントローラの向きを確認
   - USB-Cコネクタが外側（キーボードの端）向き
   - 表面（LED、チップ）が上
2. コンスルーにしっかり押し込む（カチッと音）
3. 水平性を確認
4. 左右両方実施

#### Step 5: 動作確認

1. USB-Cケーブルを接続
2. 電源スイッチをON
3. LEDが点灯するか確認 ✅

------

### Phase 3: バッテリー配線

**所要時間：30分**
 **⚠️ はんだ付け不要（JST PHコネクタで接続）**

#### バッテリー接続

1. バッテリーの極性を3回確認
   - 赤線：プラス（+）
   - 黒線：マイナス（-）
2. コントローラボードのJST PHコネクタに接続
   - カチッと音がするまで差し込む
   - 無理に力を入れない（正しい向きにスムーズに入る）
3. マスキングテープでバッテリーを軽く固定

#### 動作確認

1. 電源スイッチをON
2. USB-Cケーブル**なし**でLEDが点灯するか確認
   - 点灯した：バッテリー駆動成功 ✅
   - 点灯しない：USB接続で充電してから再確認

------

### Phase 4: FFCコネクタ接続

**所要時間：30分**
 **⚠️ はんだ付け不要（コネクタに差し込むだけ）**

#### 基板側FFCコネクタへの接続

1. FFCコネクタのツメ（黒い部分）を開ける
   - 黒い部分の左右端をつまむ
   - 上に持ち上げる（1-2mm）
2. FFCケーブルの向きを確認
   - 銅箔面（金色）をコネクタの接点側に向ける
3. ケーブルをゆっくり挿入
   - 青い補強部がコネクタの外に出る深さ
   - 銅箔部分が完全にコネクタ内に入る
4. ツメを閉じてロック
   - カチッと音がするまで押す
5. 4箇所すべて実施（左右基板各2箇所）

#### センサーモジュール側への接続

1. 14mmマウスセンサーモジュールのFFCコネクタを開ける
2. FFCケーブルの反対側を接続
3. ツメを閉じてロック
4. 2個のセンサーモジュールで実施

------

### Phase 5: トラックボール組み立て

**所要時間：1時間**

#### Step 1: セラミックボールの配置

1. ケースのトラックボール受け部分を確認
   - 3箇所の穴（120度間隔）
2. セラミックボール2mm × 3個を各穴に配置
   - ピンセットでつまんで慎重に
3. 3個のボールが同じ高さか確認
4. 左右両方で実施（合計6個使用）

#### Step 2: センサーモジュールの固定

1. センサーをトラックボール真下に配置
   - トラックボール底面との距離：1-2mm
2. 固定方法を選択
   - **方法A**：M2ネジで固定（推奨）
   - **方法B**：両面テープで固定
   - **方法C**：ホットボンド（周辺のみ）
3. FFCケーブルを整理
   - 折れ曲がらないように
   - 他の部品と干渉しないように

#### Step 3: トラックボールの設置

1. 25mmトラックボールをセラミックボール上にゆっくり乗せる
2. 3点支持で安定しているか確認
3. 指で軽く回してみる
   - スムーズに回転するか
   - 引っかかりがないか
   - 異音がしないか

------

### Phase 6: スイッチ挿入

**所要時間：30分**

1. トップケースを用意
2. ケースとPCB基板を合わせる
3. スイッチ（Kailh Mini Linear）をソケットに挿入
   - トップケースを挟みながら
   - ピンが曲がっていないか確認
   - カチッと音がするまで押し込む
4. 54個すべて挿入
5. 引っ張って抜けないか確認

------

### Phase 7: ガスケットシート貼り付け

**所要時間：15分**

#### ボトムフォーム

1. ボトムケース内面を清掃（アルコールで拭く）
2. ボトムフォームを配置
   - ケースの形状に合わせる
3. 固定方法を選択
   - **推奨**：接着なし（そのまま挟み込む）
   - **オプション**：四隅に両面テープ
4. PCB基板を乗せる

#### ガスケットフォーム

1. トップケース内側にガスケットフォームを配置
2. ケースの縁に沿わせる
3. 固定方法を選択（接着なしでもOK）

------

### Phase 8: ケース組み立て

**所要時間：30分**

1. ボトムケースにボトムフォーム + PCB基板を配置
2. トップケースを被せる
   - スイッチがケースの穴に合うか確認
3. ボトムケースの爪をトップケースに押し込む
   - 手前から順に
   - パチンと音がするまで
4. 全周確認
   - 隙間がないか
   - 爪がすべてロックされているか

**⚠️ 注意：**

- 爪が大きすぎて入らない場合：ヤスリで少し削る
- 爪が短すぎて緩い場合：手前側にマスキングテープを貼る

------

### Phase 9: 仕上げ

**所要時間：15分**

1. キーキャップを取り付け（54個）
   - カチッと音がするまで差し込む
2. 滑り止めをボトムケースに貼り付け
   - 4隅 + 中央など
   - アルコールで清掃してから貼る
3. 完成！🎉

------

### Phase 10: 最終動作確認

**所要時間：30分**

#### 全キー入力テスト

1. PCとBluetooth接続
   - ペアリングモード起動
   - PCで検出・接続
2. テキストエディタで全54キーを押す
   - すべて反応するか
   - チャタリング（誤入力）がないか

#### トラックボール動作確認

1. カーソルが動くか
   - 上下左右にスムーズか
2. トラックボールの回転確認
   - 引っかかりがないか
   - 異音がしないか

#### Bluetooth接続テスト

1. 左右通信確認
   - 右手側のキーが左手側を通じて入力されるか
2. 接続安定性
   - 途切れないか

#### バッテリー駆動確認

1. USBケーブルを外す
2. バッテリー駆動で動作確認
   - キー入力正常
   - トラックボール正常
3. 充電確認
   - USB接続時に充電LEDが点灯するか

## 💻 ファームウェア

### ZMK Firmware

WTB54はZMK Firmwareを使用しています。

#### ファームウェアのビルド

1. Auto-KDKが出力したzipファイルを展開
2. 中身をGitHubにプッシュ
   - リポジトリ名：`zmk-config-wtb54`（任意）
3. GitHub Actionsが自動的にビルド開始
4. ビルド完了後、Artifactsからuf2ファイルをダウンロード

#### ファームウェアの書き込み

1. コントローラの電源スイッチをOFF
2. USB-Cケーブルを接続
   - UF2ブートローダが起動
   - マスストレージとして認識
3. ダウンロードしたuf2ファイルをコピー
   - 左手用：`wtb54_left.uf2`
   - 右手用：`wtb54_right.uf2`
4. 自動的に書き込みが完了
5. ドライブがアンマウントされれば成功

#### キーマップの変更

**方法1：ZMK Studio（GUI）**

1. [ZMK Studio](https://zmk.studio/)にアクセス
2. Bluetooth接続
3. GUIでキーマップを編集

**方法2：ソースコード編集**

1. GitHubリポジトリの`config/wtb54.keymap`を編集
2. Commit & Push
3. GitHub Actionsで再ビルド
4. 新しいuf2ファイルを書き込み

#### トラックボール設定

左トラックボールをスクロールに設定する場合：

```conf
# config/wtb54.conf に追加

CONFIG_ZMK_MOUSE=y
CONFIG_ZMK_POINTING_DEVICE=y
```

詳細は[Auto-KDKトラックボールマニュアル](https://github.com/sekigon-gonnoc/auto-kdk/blob/master/manual/TrackBall.md)を参照。

## 🔍 トラブルシューティング

### 組み立て時

| 症状                 | 原因               | 対処法               |
| -------------------- | ------------------ | -------------------- |
| スイッチが入らない   | ソケット位置ズレ   | ソケット再実装       |
| ソケットが浮く       | はんだ不足         | 再加熱して追加はんだ |
| はんだブリッジ       | はんだ多すぎ       | 吸取線で除去         |
| コントローラ認識せず | コンスルー接触不良 | コントローラ挿し直し |

### トラックボール

| 症状                     | 原因                       | 対処法               |
| ------------------------ | -------------------------- | -------------------- |
| トラックボールが回らない | センサーに当たっている     | センサー位置を下げる |
| 回転が重い               | セラミックボール汚れ       | アルコールで清掃     |
| 引っかかる               | セラミックボール高さ不揃い | 配置を調整           |
| センサーが反応しない     | FFC接続不良                | ケーブル挿し直し     |
| カーソルが飛ぶ           | センサー距離が遠い         | センサーを近づける   |

### 電源・接続

| 症状                   | 原因                     | 対処法                     |
| ---------------------- | ------------------------ | -------------------------- |
| LEDが点灯しない        | バッテリー電圧不足       | USB接続で充電              |
| Bluetooth接続できない  | ファームウェア未書き込み | uf2ファイルを書き込み      |
| 充電できない           | バッテリー極性逆         | 配線確認（即座に外す）     |
| 一部のキーが反応しない | はんだ不良               | 該当ソケット確認・再はんだ |
